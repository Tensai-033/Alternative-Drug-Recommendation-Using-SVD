# Alternative-Drug-Recommendation-Using-SVD
A small, simple project of applying truncated SVD and cosine similarity to find alternative (not commonly used for similar purposes) drugs

### Prerequisites

This are the things you need to launch the program.
* Python 3.10 or higher
* pip
  ```sh
  python -m ensurepip --upgrade
  ```
* Jupyter Notebook
  ```sh
  pip install notebook
  ```

### Installation and Usage
1. Clone the repo
   ```sh
   git clone https://github.com/Tensai-033/Alternative-Drug-Recommendation-Using-SVD
   ```
2. Run the notebook
   ```sh
   jupyter notebook
   ```
